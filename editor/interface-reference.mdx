## Editor toolbar

The editor toolbar contains the following elements:

- **Branch selector**: Switch between branches or create new ones
- **File path**: Shows the current file location
- **Mode toggle**: Switch between visual and Markdown modes
- **Save/Publish button**: Save changes or publish to your site

## File explorer

The sidebar file explorer allows you to:

- Browse your documentation files and folders
- Create new files and folders
- Search for specific files
- Navigate your documentation structure

## Component menu

In visual mode, press <kbd>/</kbd> to access the component menu, which includes:

- Text formatting options
- Content blocks (callouts, code blocks, etc.)
- Layout components (columns, tabs, etc.)
- Interactive elements (accordions, expandables, etc.)

## Status indicators

The editor uses various indicators to show the state of your work:

- **Unsaved changes**: Indicated by a dot next to the file name
- **Branch status**: Shows current branch and any pending changes
- **Sync status**: Indicates when changes are being saved to Git
